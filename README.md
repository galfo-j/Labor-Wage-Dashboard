# 📊 PH Industry Insights: Dashboard Overview

This Streamlit application provides an interactive, data-driven platform for analyzing labor and wage trends in the Philippines using **Philippine Statistics Authority (PSA)** data.

---

## 🚀 Key Modules

* **📈 Dashboard Overview:** Displays  national indicators (Total Industries, Date Range, Avg Daily Wage) and maps the correlation between employment volumes and daily pay.
* **👥 Employment by Sector:** Visualizes workforce distribution across sectors, highlighting market share dynamics using interactive pie and bar charts.
* **💰 Wage Analysis:** Identifies the top 5 highest-paying and bottom 5 lowest-paying industries for any selected year, with multi-industry timeline comparisons.
* **🔮 Forecasting Engine:** Aggregates quarterly data into annual run-rates and uses a **Linear Regression model** to project employment trajectories 1–5 years into the future, complete with prediction intervals.
